# XTDE0860

**Type:** Dynamic error

An attribute node is created after a child node has been added to the containing element. Attributes must be created before any child content.

## Example

```xml
<div>
  <p>Some content</p>
  <xsl:attribute name="class">highlight</xsl:attribute>  <!-- XTDE0860 -->
</div>
```

## Correct Usage

```xml
<div>
  <xsl:attribute name="class">highlight</xsl:attribute>
  <p>Some content</p>
</div>
```
