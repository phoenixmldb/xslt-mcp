# XPST0017

**Type:** Static error (XPath)

A function call references a function that is not available (wrong name, wrong arity, or undefined namespace).

## Common Causes

- Typo in the function name.
- Wrong number of arguments.
- Namespace prefix not declared.
- Calling a stylesheet function from the wrong package.

## Example

```xpath
<!-- Wrong arity -->
contains("hello")  <!-- XPST0017: fn:contains requires 2 or 3 arguments -->

<!-- Unknown function -->
my:undefined-function(42)  <!-- XPST0017 (if not declared) -->
```
