# FORG0006

**Type:** Type error (XPath/XQuery Functions and Operators)

The effective boolean value is not defined for the given argument. Raised when a value is used in a boolean context but cannot be converted to a boolean.

## Common Causes

- A sequence of two or more atomic values is used where a boolean is expected.
- A function item is used in a boolean context.

## Example

```xpath
<!-- Sequence of multiple atomic values has no EBV -->
if ((1, 2)) then "yes" else "no"  <!-- FORG0006 -->

<!-- Single values are fine -->
if (1) then "yes" else "no"  <!-- OK: true -->
if (()) then "yes" else "no"  <!-- OK: false -->
```
