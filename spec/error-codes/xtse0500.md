# XTSE0500

**Type:** Static error

The value of the `match` attribute on `xsl:template` is not a valid pattern.

## Common Causes

- Syntax error in the match pattern.
- Using an XPath expression that is not valid as a pattern.

## Example

```xml
<!-- Invalid pattern syntax -->
<xsl:template match="[invalid">
  <!-- XTSE0500 -->
</xsl:template>
```
