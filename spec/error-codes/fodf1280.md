# FODF1280

**Type:** Dynamic error (XPath/XQuery Functions and Operators)

The picture string supplied to `fn:format-number` is invalid.

## Common Causes

- Invalid format characters.
- Malformed grouping patterns.
- Missing mandatory digit in sub-picture.

## Example

```xpath
format-number(42, '##')  <!-- FODF1280: no mandatory digit -->
format-number(42, '#.#.#')  <!-- FODF1280: multiple decimal separators -->
```
