# XTTE0505

**Type:** Type error

The result of a template, function, or variable does not match the required type declared in the `as` attribute.

## Example

```xml
<xsl:template match="item" as="element(result)">
  <!-- Returns a string, not an element — XTTE0505 -->
  <xsl:value-of select="@name"/>
</xsl:template>

<xsl:function name="my:get-count" as="xs:integer">
  <!-- Returns a string — XTTE0505 -->
  <xsl:sequence select="'not a number'"/>
</xsl:function>
```
