# XPST0008

**Type:** Static error (XPath)

An expression references a variable that is not in scope, or an undeclared namespace prefix is used.

## Common Causes

- Typo in a variable name.
- Referencing a variable before it is declared.
- Using an undeclared namespace prefix.

## Example

```xml
<xsl:value-of select="$undeclared-variable"/>  <!-- XPST0008 -->

<xsl:value-of select="unknown:element"/>  <!-- XPST0008: prefix 'unknown' not declared -->
```
