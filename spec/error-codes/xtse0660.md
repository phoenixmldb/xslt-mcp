# XTSE0660

**Type:** Static error

A required parameter of a template is not supplied by `xsl:with-param` in the calling instruction.

## Example

```xml
<xsl:template name="greet">
  <xsl:param name="name" required="yes"/>
  Hello, <xsl:value-of select="$name"/>!
</xsl:template>

<xsl:call-template name="greet"/>
<!-- XTSE0660: required parameter 'name' not supplied -->
```
