# XTTE0590

**Type:** Type error

The supplied value for a parameter does not match the required type declared in the `as` attribute.

## Example

```xml
<xsl:template name="show">
  <xsl:param name="count" as="xs:integer"/>
  <xsl:value-of select="$count"/>
</xsl:template>

<xsl:call-template name="show">
  <xsl:with-param name="count" select="'not-a-number'"/>
  <!-- XTTE0590 -->
</xsl:call-template>
```
