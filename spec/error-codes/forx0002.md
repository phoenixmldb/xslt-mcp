# FORX0002

**Type:** Dynamic error (XPath/XQuery Functions and Operators)

The regular expression is invalid. Raised by `fn:matches`, `fn:replace`, `fn:tokenize`, and `fn:analyze-string` when the pattern cannot be compiled.

## Example

```xpath
matches("hello", "[invalid") <!-- FORX0002: unclosed character class -->
replace("text", "(unclosed", "x") <!-- FORX0002 -->
```
