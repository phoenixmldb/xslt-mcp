# XTSE0020

**Type:** Static error

The `version` attribute on `xsl:stylesheet` (or `xsl:transform`) is required but missing, or the value is not a valid version number.

## Common Causes

- Forgot to include `version="3.0"` on the stylesheet element.
- Used an unrecognized version number.

## Example

```xml
<!-- Missing version attribute -->
<xsl:stylesheet xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <!-- XTSE0020 -->
</xsl:stylesheet>
```
