# FODC0002

**Type:** Dynamic error (XPath/XQuery Functions and Operators)

A document URI cannot be retrieved. Typically raised by `fn:doc()` or `fn:document()` when the specified resource is not available.

## Common Causes

- File not found.
- Network resource unreachable.
- Invalid URI.
- Access denied.

## Example

```xpath
doc('nonexistent.xml')  <!-- FODC0002 -->
document('http://unreachable.example.com/data.xml')  <!-- FODC0002 -->
```
