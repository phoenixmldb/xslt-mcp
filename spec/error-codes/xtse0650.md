# XTSE0650

**Type:** Static error

An `xsl:call-template` instruction references a template name that does not exist in the stylesheet.

## Example

```xml
<xsl:call-template name="nonexistent-template"/>
<!-- XTSE0650: no template named 'nonexistent-template' -->
```
