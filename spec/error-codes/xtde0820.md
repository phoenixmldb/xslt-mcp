# XTDE0820

**Type:** Dynamic error

The effective value of the `name` attribute of `xsl:element` is not a valid QName.

## Example

```xml
<xsl:variable name="bad-name" select="'123invalid'"/>
<xsl:element name="{$bad-name}">  <!-- XTDE0820 at runtime -->
  content
</xsl:element>
```
