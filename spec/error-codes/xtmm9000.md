# XTMM9000

**Type:** Dynamic error

The default error code raised when `xsl:message terminate="yes"` is used without specifying a custom `error-code`.

## Example

```xml
<xsl:message terminate="yes">
  Fatal: input data is invalid
</xsl:message>
<!-- Raises XTMM9000 and terminates -->
```
