# XTSE0740

**Type:** Static error

The `name` attribute of `xsl:function` is not a prefixed QName (the function must be in a non-null namespace).

## Example

```xml
<!-- Function name must be in a namespace -->
<xsl:function name="my-func">  <!-- XTSE0740 -->
  <xsl:sequence select="42"/>
</xsl:function>

<!-- Correct: use a prefixed name -->
<xsl:function name="my:my-func" xmlns:my="http://example.com/my">
  <xsl:sequence select="42"/>
</xsl:function>
```
