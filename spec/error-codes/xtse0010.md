# XTSE0010

**Type:** Static error

A general static error indicating that the stylesheet does not conform to the XSLT specification. This is a catch-all error code for structural violations.

## Common Causes

- A required attribute is missing (e.g., `test` on `xsl:if`).
- An attribute has an invalid value.
- An element appears in an invalid position (e.g., `xsl:otherwise` not last in `xsl:choose`).
- Content is not permitted (e.g., child elements inside `xsl:text`).
- A top-level element is not a valid declaration.

## Examples

```xml
<!-- Missing required 'test' attribute -->
<xsl:if>
  <xsl:text>This will cause XTSE0010</xsl:text>
</xsl:if>

<!-- xsl:otherwise not last -->
<xsl:choose>
  <xsl:otherwise/>
  <xsl:when test="true()"/>  <!-- XTSE0010 -->
</xsl:choose>
```
