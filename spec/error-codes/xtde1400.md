# XTDE1400

**Type:** Dynamic error

Two calls to `xsl:result-document` produce output with the same effective URI. Each result document must have a unique URI.

## Example

```xml
<xsl:for-each select="item">
  <!-- If two items have the same @id, XTDE1400 -->
  <xsl:result-document href="{@id}.html">
    <xsl:apply-templates select="."/>
  </xsl:result-document>
</xsl:for-each>
```
