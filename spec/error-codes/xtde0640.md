# XTDE0640

**Type:** Dynamic error

A circular definition is detected among variables and/or parameters. Variable A depends on variable B which depends on variable A.

## Example

```xml
<xsl:variable name="a" select="$b + 1"/>
<xsl:variable name="b" select="$a + 1"/>
<!-- XTDE0640: circular dependency -->
```
