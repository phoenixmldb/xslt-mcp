# XTSE1080

**Type:** Static error

An `xsl:for-each-group` instruction does not have exactly one grouping attribute. Exactly one of `group-by`, `group-adjacent`, `group-starting-with`, or `group-ending-with` must be specified.

## Example

```xml
<!-- Missing grouping attribute -->
<xsl:for-each-group select="item">  <!-- XTSE1080 -->
  <xsl:value-of select="."/>
</xsl:for-each-group>

<!-- Two grouping attributes -->
<xsl:for-each-group select="item" group-by="@cat" group-adjacent="@cat">  <!-- XTSE1080 -->
  <xsl:value-of select="."/>
</xsl:for-each-group>
```
