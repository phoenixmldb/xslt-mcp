# XTDE0850

**Type:** Dynamic error

The effective value of the `name` attribute of `xsl:attribute` is not a valid QName, or is `xmlns`.

## Example

```xml
<xsl:attribute name="{$computed-name}">value</xsl:attribute>
<!-- XTDE0850 if $computed-name is not a valid QName -->
```
