# XTSE0090

**Type:** Static error

An `xsl:import` element is not preceding all other children of `xsl:stylesheet`. All `xsl:import` elements must appear before any other top-level declarations.

## Example

```xml
<xsl:stylesheet version="3.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform">
  <xsl:template match="/"/>
  <xsl:import href="other.xsl"/>  <!-- XTSE0090: import must come first -->
</xsl:stylesheet>
```
